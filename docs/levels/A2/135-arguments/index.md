# Streit / Arguments
---

## 001: Über Streit sprechen

**Situation:**  
Zwei Freunde sprechen über einen Streit.  
*Two friends talk about an argument.*

### Dialog / Dialogue

**Anna:**  
Ich habe mich mit meinem Freund gestritten.  
*I've argued with my boyfriend.*

**Tom:**  
Das tut mir leid! Worum ging es?  
*I'm sorry! What was it about?*

**Anna:**  
Um eine Kleinigkeit.  
*About something small.*

**Tom:**  
Das passiert manchmal.  
*That happens sometimes.*

**Anna:**  
Ja, aber es war unangenehm.  
*Yes, but it was unpleasant.*

**Tom:**  
Das kann ich verstehen!  
*I can understand that!*

**Anna:**  
Ja, wir haben beide Fehler gemacht.  
*Yes, we both made mistakes.*

**Tom:**  
Das ist normal!  
*That's normal!*

**Anna:**  
Ja, aber jetzt müssen wir darüber reden.  
*Yes, but now we need to talk about it.*

*That's true!*

*Yes, I hope we can clarify it.*

## Grammatik / Grammar

### Perfekt Tense

**English:** The Perfekt tense describes completed actions in the past. It's formed with haben or sein + past participle.

**Examples from conversations:**

- **Ich habe mich gestritten.** - I've argued.
- **Wir haben unsere Fehler erkannt.** - We recognized our mistakes.
- **Wir haben uns entschuldigt.** - We apologized.

### Modal Verbs (können, sollen, müssen)

**English:** Modal verbs express ability, obligation, or necessity. They are conjugated and the main verb goes to the end in infinitive form.

**Examples from conversations:**

- **Wir können den Streit lösen.** - We can solve the argument.
- **Man sollte miteinander reden.** - You should talk with each other.
- **Man muss Kompromisse machen.** - You have to make compromises.

### Subordinate Clauses with "dass" and "weil"

**English:** Subordinate clauses introduced by "dass" (that) or "weil" (because) push the verb to the end.

**Examples from conversations:**

- **Ich hoffe, dass wir es klären können.** - I hope that we can clarify it.
- **Weil wir nicht kommuniziert haben, gab es einen Streit.** - Because we didn't communicate, there was an argument.
- **Ich finde, dass Kommunikation wichtig ist.** - I think that communication is important.

### Reflexive Verbs

**English:** Reflexive verbs use reflexive pronouns (mich, dich, sich, uns, euch) that match the subject.

**Examples from conversations:**

- **Wir haben uns gestritten.** - We argued.
- **Wir sollten miteinander reden.** - We should talk with each other.
- **Wir haben uns entschuldigt.** - We apologized.
