# Projekte / Projects

---

## 001: Projekt planen / Planning a Project

**Situation:**  
Lisa plant ein Projekt mit ihrem Team.  
*Lisa plans a project with her team.*

### Dialog / Dialogue

**Lisa:**  
Lass uns das neue Projekt planen.  
*Let's plan the new project.*

**Teammitglied:**  
Gute Idee! Was ist das Ziel?  
*Good idea! What's the goal?*

**Lisa:**  
Wir wollen eine neue Marketingkampagne entwickeln.  
*We want to develop a new marketing campaign.*

**Teammitglied:**  
Wie lange haben wir Zeit?  
*How long do we have?*

**Lisa:**  
Drei Monate. Das sollte ausreichen.  
*Three months. That should be enough.*

**Teammitglied:**  
Was sind die ersten Schritte?  
*What are the first steps?*

**Lisa:**  
Zuerst müssen wir recherchieren und dann einen Plan machen.  
*First we need to research and then make a plan.*

**Teammitglied:**  
Das klingt gut. Lass uns anfangen!  
*That sounds good. Let's start!*

## 002: Projekt leiten / Leading a Project

**Situation:**  
Max leitet ein Projekt.  
*Max leads a project.*

### Dialog / Dialogue

**Max:**  
Ich leite dieses Projekt.  
*I'm leading this project.*

**Teammitglied:**  
Was sind deine Aufgaben?  
*What are your tasks?*

**Max:**  
Ich koordiniere das Team, plane die Aufgaben und überwache den Fortschritt.  
*I coordinate the team, plan the tasks and monitor the progress.*

**Teammitglied:**  
Das klingt nach viel Verantwortung.  
*That sounds like a lot of responsibility.*

**Max:**  
Ja, aber ich mag es, Projekte zu leiten.  
*Yes, but I like leading projects.*

**Teammitglied:**  
Wie läuft das Projekt bisher?  
*How is the project going so far?*

**Max:**  
Sehr gut! Wir sind im Zeitplan.  
*Very well! We're on schedule.*

## Grammatik / Grammar

### Project Language: "Projekt planen", "Projekt leiten", "Projekt abschließen"

**English:** These expressions are used when discussing projects.

**Examples from conversations:**

- **Wir planen das Projekt.** - We plan the project.
- **Ich leite das Projekt.** - I lead the project.
- **Wir schließen das Projekt ab.** - We complete the project.

### Subordinate Clauses with "weil", "dass", and "wenn"

**English:** "Weil" introduces reasons, "dass" introduces statements, and "wenn" introduces conditions.

**Examples from conversations:**

- **Weil wir effizient arbeiten.** - Because we work efficiently.
- **Ich denke, dass gute Planung wichtig ist.** - I think that good planning is important.
- **Wenn wir Risiken früh erkennen.** - When we recognize risks early.

### Modal Verbs: können, sollen, müssen

**English:** Modal verbs are used when discussing project tasks and requirements.

**Examples from conversations:**

- **Wir können das schaffen.** - We can do that.
- **Wir sollten Prioritäten setzen.** - We should set priorities.
- **Wir müssen die Aufgaben erledigen.** - We must complete the tasks.

### Perfekt Tense - Project Completion

**English:** The Perfekt tense is used to describe completed projects and achievements.

**Examples from conversations:**

- **Wir haben das Projekt abgeschlossen.** - We've completed the project.
- **Wir haben alle Ziele erreicht.** - We've achieved all goals.
- **Ich habe viel gelernt.** - I learned a lot.
