# Internetanschluss / Internet Setup

---

## 001: Internet bestellen / Ordering Internet

**Situation:**  
Anna bestellt einen Internetanschluss.  
*Anna orders an internet connection.*

### Dialog / Dialogue

**Anna:**  
Ich brauche einen Internetanschluss.  
*I need an internet connection.*

**Mitarbeiter:**  
Welche Geschwindigkeit möchten Sie?  
*What speed would you like?*

**Anna:**  
Was empfehlen Sie?  
*What do you recommend?*

**Mitarbeiter:**  
Für normale Nutzung reichen 50 Mbit/s.  
*For normal use, 50 Mbit/s is enough.*

**Anna:**  
Das klingt gut.  
*That sounds good.*

**Mitarbeiter:**  
Wann möchten Sie den Anschluss?  
*When would you like the connection?*

**Anna:**  
So schnell wie möglich.  
*As soon as possible.*

**Mitarbeiter:**  
Ich schaue, wann ein Termin frei ist.  
*I'll check when an appointment is available.*

**Anna:**  
Danke, das wäre super.  
*Thanks, that would be great.*

## Grammatik / Grammar

### Modal Verbs (müssen, können, möchten)

**English:** Modal verbs express necessity, ability, or desire. They are used with another verb in the infinitive form.

**Examples from conversations:**

- **Ich brauche einen Internetanschluss** - I need an internet connection
- **Wie verbinde ich mein Handy?** - How do I connect my phone?
- **Ich möchte meinen Vertrag kündigen** - I want to cancel my contract
- **Möchten Sie verlängern?** - Would you like to renew?

### Perfekt (Past Tense)

**English:** The Perfekt tense describes completed actions in the past, formed with "haben" or "sein" + past participle.

**Examples from conversations:**

- **Hast du den Router neu gestartet?** - Have you restarted the router?
- **Seit gestern** - Since yesterday
- **Was ist passiert?** - What happened?

### Dative Case (indirect objects and prepositions)

**English:** The dative case is used for indirect objects and after certain prepositions.

**Examples from conversations:**

- **Ich schicke Ihnen eine Bestätigung** - I'll send you a confirmation
- **Danke für die Hilfe** - Thanks for the help
- **Im Wohnzimmer** - In the living room

### Temporal Expressions (seit, wann, bald)

**English:** Temporal expressions indicate time relationships: "seit" (since), "wann" (when), "bald" (soon).

**Examples from conversations:**

- **Seit heute Morgen** - Since this morning
- **Wann kann er kommen?** - When can he come?
- **Mein Vertrag läuft bald ab** - My contract expires soon
- **In einem Monat** - In one month
