# Berufliche Weiterentwicklung / Professional Development

---

## 001: Weiterbildungsplan erstellen / Creating Development Plan

**Situation:**  
Anna erstellt einen Weiterbildungsplan mit ihrem Chef.  
*Anna creates a development plan with her boss.*

### Dialog / Dialogue

**Anna:**  
Ich möchte einen Weiterbildungsplan erstellen.  
*I'd like to create a development plan.*

**Chef:**  
Das ist eine gute Idee.  
*That's a good idea.*

**Anna:**  
Welche Möglichkeiten gibt es?  
*What possibilities are there?*

**Chef:**  
Wir können verschiedene Trainings anbieten.  
*We can offer various trainings.*

**Anna:**  
Das klingt interessant.  
*That sounds interesting.*

**Chef:**  
Was möchten Sie lernen?  
*What would you like to learn?*

**Anna:**  
Ich interessiere mich für Projektmanagement.  
*I'm interested in project management.*

**Chef:**  
Das können wir unterstützen.  
*We can support that.*

**Anna:**  
Das wäre großartig.  
*That would be great.*

## Grammatik / Grammar

### Modal Verbs (möchten, können, sollen)

**English:** Modal verbs express wishes, ability, or obligation. They are used with another verb in the infinitive form.

**Examples from conversations:**

- **Ich möchte einen Weiterbildungsplan erstellen** - I'd like to create a development plan
- **Wir können verschiedene Trainings anbieten** - We can offer various trainings
- **Sie sollten regelmäßig üben** - You should practice regularly
- **Ich möchte eine Führungsposition erreichen** - I'd like to reach a management position

### Perfekt (Past Tense)

**English:** The Perfekt tense describes completed actions in the past, formed with "haben" or "sein" + past participle.

**Examples from conversations:**

- **Ich habe ein Zertifikat erhalten** - I've received a certificate
- **Ich habe einen Kurs besucht** - I attended a course
- **Ich habe viele Tipps bekommen** - I got many tips
- **Ich habe Verbesserungen bemerkt** - I've noticed improvements

### Future Tense with "werden"

**English:** The future tense is formed with "werden" + infinitive to express future actions or intentions.

**Examples from conversations:**

- **Ich werde ein Training besuchen** - I'll attend training
- **Ich werde mich weiterentwickeln** - I'll continue developing
- **Das wird hilfreich sein** - That will be helpful

### Subordinate Clauses with "dass" and "weil"

**English:** Subordinate clauses provide additional information and are introduced by conjunctions like "dass" (that) or "weil" (because).

**Examples from conversations:**

- **Ich hoffe, dass es klappt** - I hope that it works out
- **Ich lerne viel, weil ich regelmäßig übe** - I learn a lot because I practice regularly
