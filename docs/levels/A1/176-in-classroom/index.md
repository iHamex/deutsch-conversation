# Im Klassenzimmer / In Classroom

---

## 001: Im Klassenzimmer ankommen / Arriving in Classroom

**Situation:**  
Anna kommt im Klassenzimmer an.  
*Anna arrives in the classroom.*

### Dialog / Dialogue

**Anna:**  
Guten Morgen!  
*Good morning!*

**Lehrer:**  
Guten Morgen! Setzen Sie sich bitte.  
*Good morning! Please sit down.*

**Anna:**  
Danke!  
*Thanks!*

**Lehrer:**  
Wo möchten Sie sitzen?  
*Where would you like to sit?*

**Anna:**  
Hier vorne, bitte.  
*Here in front, please.*

**Lehrer:**  
Sehr gut.  
*Very good.*

**Anna:**  
Wann beginnt der Unterricht?  
*When does the class start?*

**Lehrer:**  
Gleich, in fünf Minuten.  
*Soon, in five minutes.*

*Page twenty.*

**Tom:**  
Verstanden.  
*Understood.*

**Lehrer:**  
Haben Sie die Seite gefunden?  
*Did you find the page?*

**Tom:**  
Ja, ich habe sie gefunden.  
*Yes, I found it.*

**Lehrer:**  
Sehr gut.  
*Very good.*

## 003: Frage stellen / Asking Question

**Situation:**  
Maria stellt eine Frage.  
*Maria asks a question.*

### Dialog / Dialogue

**Maria:**  
Entschuldigung, ich habe eine Frage.  
*Excuse me, I have a question.*

**Lehrer:**  
Ja, bitte stellen Sie Ihre Frage.  
*Yes, please ask your question.*

**Maria:**  
Was bedeutet dieses Wort?  
*What does this word mean?*

**Lehrer:**  
Das bedeutet "Haus".  
*That means "house".*

**Maria:**  
Verstanden, danke!  
*Understood, thanks!*

**Lehrer:**  
Gern geschehen.  
*You're welcome.*

**Maria:**  
Kann ich noch eine Frage stellen?  
*Can I ask another question?*

**Lehrer:**  
Ja, natürlich.  
*Yes, of course.*

*Yes, I'm coming.*

**Lehrer:**

*Please write this sentence.*

**Sarah:**

**Lehrer:**

Okay, ich schreibe es.  
*Okay, I'm writing it.*

**Lehrer:**  
Sehr gut!  
*Very good!*

**Lisa:**

## Grammatik / Grammar

### Expressing "please" with "bitte" (please)

**English:** To make polite requests in the classroom, use "bitte" (please) with commands. This is essential for polite communication with teachers.

**Examples from conversations:**

- **Setzen Sie sich bitte** - Please sit down
- **Bitte öffnen Sie Ihre Bücher** - Please open your books
- **Bitte stellen Sie Ihre Frage** - Please ask your question
- **Schreiben Sie bitte diesen Satz** - Please write this sentence

### Asking "which" with "welche" (which)

**English:** To ask which page, exercise, or item, use "welche" (which) followed by the noun. This helps identify specific items in class.

**Examples from conversations:**

- **Auf welcher Seite?** - On which page?
- **Welchen Satz?** - Which sentence?
- **Welche Übungen?** - Which exercises?
- **Mit wem soll ich arbeiten?** - Who should I work with?

### Expressing "I can" with "ich kann" (I can)

**English:** To express that you can do something, use "ich kann" (I can) followed by the verb. This shows ability or willingness to participate.

**Examples from conversations:**

- **Ich kann es versuchen** - I can try
- **Wer kann diese Frage beantworten?** - Who can answer this question?
- **Kann ich noch eine Frage stellen?** - Can I ask another question?
- **Ja, natürlich** - Yes, of course

### Expressing "we do" with "wir machen" (we do)

**English:** To express that the class is doing something together, use "wir machen" (we do) followed by the activity. This describes classroom activities.

**Examples from conversations:**

- **Jetzt machen wir Partnerarbeit** - Now we're doing partner work
- **Jetzt machen wir Gruppenarbeit** - Now we're doing group work
- **Heute schreiben wir einen Test** - Today we're writing a test
- **Wie viele Personen pro Gruppe?** - How many people per group?
