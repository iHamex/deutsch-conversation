# Am Kiosk / At the Kiosk

---

## 001: Zeitung kaufen / Buying a Newspaper

**Situation:**  
Julia kauft eine Zeitung am Kiosk.  
*Julia buys a newspaper at the kiosk.*

### Dialog / Dialogue

**Julia:**  
Guten Tag. Ich möchte eine Zeitung.  
*Good day. I'd like a newspaper.*

**Kioskbesitzer:**  
Gut. Welche Zeitung möchten Sie?  
*Good. What newspaper would you like?*

**Julia:**  
Die Tageszeitung, bitte.  
*The daily newspaper, please.*

**Kioskbesitzer:**  
Sehr gut. Das kostet zwei Euro.  
*Very good. That costs two euros.*

**Julia:**  
Hier ist das Geld.  
*Here's the money.*

**Kioskbesitzer:**  
Danke schön.  
*Thank you very much.*

**Julia:**  
Gern geschehen.  
*You're welcome.*

## Grammatik / Grammar

### Expressing Wants with "Ich möchte"

**English:** Use "Ich möchte..." (I'd like...) to express what you want to buy at the kiosk. This is polite and appropriate for A1 level.

**Examples from conversations:**

- **Ich möchte eine Zeitung** - I'd like a newspaper
- **Ich möchte eine Packung Zigaretten** - I'd like a pack of cigarettes
- **Ich möchte Süßigkeiten** - I'd like sweets
- **Ich möchte ein Getränk** - I'd like a drink

### Kiosk Items Vocabulary

**English:** Common items sold at kiosks include "Zeitung" (newspaper), "Zigaretten" (cigarettes), "Süßigkeiten" (sweets), "Getränk" (drink), "Zeitschrift" (magazine), "Postkarte" (postcard), and "Kaugummi" (gum).

**Examples from conversations:**

- **Ich möchte eine Zeitung** - I'd like a newspaper
- **Ich möchte eine Packung Zigaretten** - I'd like a pack of cigarettes
- **Ich möchte Schokolade** - I'd like chocolate
- **Ich möchte eine Cola** - I'd like a cola

### Asking "What brand/variety?" with "Welche"

**English:** Use "Welche Marke möchten Sie?" (What brand would you like?) or "Welche Sorte möchten Sie?" (What variety would you like?) to ask about specific preferences.

**Examples from conversations:**

- **Welche Zeitung möchten Sie?** - What newspaper would you like?
- **Welche Marke möchten Sie?** - What brand would you like?
- **Welche Süßigkeiten möchten Sie?** - What sweets would you like?
- **Welche Sorte möchten Sie?** - What variety would you like?

### Expressing Totals

**English:** When buying multiple items, use "Das macht zusammen..." (That's... together) to state the total.

**Examples from conversations:**

- **Das macht zusammen fünf Euro fünfzig** - That's five euros fifty together
- **Das macht zusammen vier Euro** - That's four euros together
- **Ich möchte eine Zeitung, eine Cola und eine Schokolade** - I'd like a newspaper, a cola and a chocolate
