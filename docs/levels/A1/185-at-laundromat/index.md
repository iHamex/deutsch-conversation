# Im Waschsalon / At the Laundromat

---

## 001: Den Waschsalon finden / Finding the Laundromat

**Situation:**  
Anna sucht einen Waschsalon und fragt einen Passanten.  
*Anna is looking for a laundromat and asks a passerby.*

### Dialog / Dialogue

**Anna:**  
Entschuldigung! Wo ist der nächste Waschsalon?  
*Excuse me! Where is the nearest laundromat?*

**Passant:**  
Der Waschsalon ist dort hinten, links.  
*The laundromat is back there, on the left.*

**Anna:**  
Wie weit ist es?  
*How far is it?*

**Passant:**  
Etwa fünf Minuten zu Fuß.  
*About five minutes on foot.*

**Anna:**  
Danke schön!  
*Thank you very much!*

**Passant:**  
Gern geschehen!  
*You're welcome!*

---

## 002: Eine Waschmaschine suchen / Looking for a Washing Machine

**Situation:**  
Tom kommt in den Waschsalon und sucht eine freie Waschmaschine.  
*Tom comes into the laundromat and looks for a free washing machine.*

### Dialog / Dialogue

**Tom:**  
Guten Tag! Sind alle Waschmaschinen belegt?  
*Good day! Are all washing machines occupied?*

**Angestellter:**  
Nein, Maschine Nummer drei ist frei.  
*No, machine number three is free.*

**Tom:**  
Gut! Wie funktioniert sie?  
*Good! How does it work?*

**Angestellter:**  
Sie werfen Münzen ein und wählen ein Programm.  
*You insert coins and select a program.*

**Tom:**  
Welche Programme gibt es?  
*Which programs are there?*

**Angestellter:**  
Normal, Feinwäsche und Kochwäsche.  
*Normal, delicates, and boil wash.*

**Tom:**  
Verstanden. Danke!  
*Understood. Thanks!*

## 003: Wäsche waschen / Washing Laundry

**Situation:**  
Sophie wäscht ihre Wäsche im Waschsalon.  
*Sophie washes her laundry at the laundromat.*

### Dialog / Dialogue

**Sophie:**  
Ich möchte meine Wäsche waschen.  
*I would like to wash my laundry.*

**Angestellte:**  
Gut! Welche Maschine möchten Sie nehmen?  
*Good! Which machine would you like to use?*

**Sophie:**  
Maschine Nummer fünf, bitte.  
*Machine number five, please.*

**Angestellte:**  
Gut. Welches Programm?  
*Good. Which program?*

**Sophie:**  
Das normale Programm für bunte Wäsche.  
*The normal program for colored laundry.*

**Angestellte:**  
Gut. Das kostet drei Euro.  
*Good. That costs three euros.*

**Sophie:**  
Hier sind drei Euro.  
*Here are three euros.*

**Angestellte:**  
Danke! Die Maschine läuft jetzt.  
*Thanks! The machine is running now.*

## 004: Warten auf die Wäsche / Waiting for the Laundry

**Situation:**  
David wartet, während seine Wäsche gewaschen wird.  
*David waits while his laundry is being washed.*

### Dialog / Dialogue

**David:**  
Wie lange dauert das Waschen?  
*How long does the washing take?*

**Angestellter:**  
Etwa dreißig Minuten.  
*About thirty minutes.*

**David:**  
Kann ich hier warten?  
*Can I wait here?*

**Angestellter:**  
Ja, natürlich! Es gibt Stühle dort.  
*Yes, of course! There are chairs over there.*

**David:**  
Gut, danke!  
*Good, thanks!*

**Angestellter:**  
Gern geschehen!  
*You're welcome!*

**David:**  
Gibt es hier eine Zeitung?  
*Is there a newspaper here?*

**Angestellter:**  
Ja, dort auf dem Tisch.  
*Yes, over there on the table.*

## 005: Wäsche in den Trockner tun / Putting Laundry in the Dryer

**Situation:**  
Emma nimmt ihre Wäsche aus der Waschmaschine und tut sie in den Trockner.  
*Emma takes her laundry out of the washing machine and puts it in the dryer.*

### Dialog / Dialogue

**Emma:**  
Meine Wäsche ist fertig.  
*My laundry is finished.*

**Angestellte:**  
Gut! Möchten Sie die Wäsche trocknen?  
*Good! Would you like to dry the laundry?*

**Emma:**  
Ja, bitte! Wo ist der Trockner?  
*Yes, please! Where is the dryer?*

**Angestellte:**  
Die Trockner sind dort hinten.  
*The dryers are back there.*

**Emma:**  
Welchen Trockner soll ich nehmen?  
*Which dryer should I use?*

**Angestellte:**  
Trockner Nummer zwei ist frei.  
*Dryer number two is free.*

**Emma:**  
Gut! Wie viel kostet das Trocknen?  
*Good! How much does drying cost?*

**Angestellte:**  
Zwei Euro für dreißig Minuten.  
*Two euros for thirty minutes.*

**Emma:**  
Verstanden.  
*Understood.*

## 006: Probleme mit der Maschine / Problems with the Machine

**Situation:**  
Felix hat ein Problem mit der Waschmaschine.  
*Felix has a problem with the washing machine.*

### Dialog / Dialogue

**Felix:**  
Entschuldigung! Die Maschine funktioniert nicht.  
*Excuse me! The machine doesn't work.*

**Angestellter:**  
Welche Maschine?  
*Which machine?*

**Felix:**  
Maschine Nummer vier.  
*Machine number four.*

**Angestellter:**  
Moment, ich schaue nach.  
*One moment, let me check.*

**Felix:**  
Ich habe schon Münzen eingeworfen.  
*I already inserted coins.*

**Angestellter:**  
Das tut mir leid. Sie können eine andere Maschine nehmen.  
*I'm sorry. You can use another machine.*

**Felix:**  
Gut, dann nehme ich Maschine Nummer sechs.  
*Good, then I'll use machine number six.*

**Angestellter:**  
Gut, die ist frei.  
*Good, that one is free.*

**Felix:**  
Danke!  
*Thanks!*

## 008: Wäsche falten / Folding Laundry

Nina faltet ihre trockene Wäsche.

*Good! Would you like to fold the laundry here?*

**Nina:**  
Ja, gibt es einen Tisch dafür?  
*Yes, is there a table for that?*

**Angestellte:**  
Ja, dort hinten ist ein großer Tisch.  
*Yes, back there is a large table.*

**Nina:**  
Perfekt! Danke!  
*Perfect! Thanks!*

**Angestellte:**  
Gern geschehen!  
*You're welcome!*

**Nina:**  
Ich falte die Wäsche jetzt.  
*I'm folding the laundry now.*

## 010: Über den Waschsalon sprechen / Talking About the Laundromat

**Situation:**  
Paul erzählt seinem Freund Jonas vom Waschsalon.  
*Paul tells his friend Jonas about the laundromat.*

### Dialog / Dialogue

**Paul:**  
Hallo Jonas! Ich war heute im Waschsalon.  
*Hello Jonas! I was at the laundromat today.*

**Jonas:**  
Wie war es?  
*How was it?*

**Paul:**  
Gut! Die Maschinen sind modern.  
*Good! The machines are modern.*

**Jonas:**  
Wie viel kostet das Waschen?  
*How much does washing cost?*

**Paul:**  
Drei Euro für eine normale Wäsche.  
*Three euros for a normal wash.*

**Jonas:**  
Das ist ein guter Preis.  
*That's a good price.*

**Paul:**  
Ja, und der Trockner kostet zwei Euro.  
*Yes, and the dryer costs two euros.*

**Jonas:**  
Das ist auch günstig.  
*That's also cheap.*

**Paul:**  
Ja, ich bin zufrieden!  
*Yes, I'm satisfied!*

## Grammatik / Grammar

### Modal Verb "möchten" (would like)

**English:** "Möchten" expresses polite wishes or wants. It is conjugated and the main verb goes to the end.

**Examples from conversations:**

- **Ich möchte meine Wäsche waschen.** - I would like to wash my laundry.
- **Möchten Sie die Wäsche trocknen?** - Would you like to dry the laundry?
- **Möchten Sie die Wäsche hier falten?** - Would you like to fold the laundry here?

### Asking About Price (wie viel kostet)

**English:** "Wie viel kostet" means "how much does it cost". It's used to ask about prices.

**Examples from conversations:**

- **Wie viel kostet das Waschen?** - How much does washing cost?
- **Wie viel kostet das Trocknen?** - How much does drying cost?
- **Das kostet drei Euro.** - That costs three euros.

### Expressing Duration (wie lange dauert)

**English:** "Wie lange dauert" means "how long does it take". "Dauern" means "to last" or "to take".

**Examples from conversations:**

- **Wie lange dauert das Waschen?** - How long does the washing take?
- **Etwa dreißig Minuten.** - About thirty minutes.
- **Zwei Euro für dreißig Minuten.** - Two euros for thirty minutes.

### Questions with "welche" (which)

**English:** "Welche" is used to ask about choices among multiple options. It changes form based on gender and number.

**Examples from conversations:**

- **Welche Maschine möchten Sie nehmen?** - Which machine would you like to use?
- **Welches Programm?** - Which program?
- **Welchen Trockner soll ich nehmen?** - Which dryer should I use?
