# Über das Wetter sprechen / Talking About the Weather

---

## 001: Wie ist das Wetter? / How Is the Weather?

**Situation:**  
Zwei Freunde sprechen über das Wetter.  
*Two friends talk about the weather.*

### Dialog / Dialogue

**Anna:**  
Wie ist das Wetter heute?  
*How is the weather today?*

**Tom:**  
Es ist sonnig.  
*It's sunny.*

**Anna:**  
Das ist schön!  
*That's nice!*

**Tom:**  
Ja, sehr schön!  
*Yes, very nice!*

**Anna:**  
Wie warm ist es?  
*How warm is it?*

**Tom:**  
Es ist zwanzig Grad.  
*It's twenty degrees.*

**Anna:**  
Das ist perfekt!  
*That's perfect!*

## 002: Regenwetter / Rainy Weather

**Situation:**  
Zwei Kollegen sprechen über das Regenwetter.  
*Two colleagues talk about the rainy weather.*

### Dialog / Dialogue

**Herr Schmidt:**  
Guten Morgen! Wie ist das Wetter?  
*Good morning! How is the weather?*

**Frau Müller:**  
Es regnet.  
*It's raining.*

**Herr Schmidt:**  
Oh nein! Haben Sie einen Regenschirm?  
*Oh no! Do you have an umbrella?*

**Frau Müller:**  
Ja, ich habe einen Regenschirm.  
*Yes, I have an umbrella.*

**Herr Schmidt:**  
Das ist gut!  
*That's good!*

**Frau Müller:**  
Ja, aber es regnet sehr stark.  
*Yes, but it's raining very hard.*

**Herr Schmidt:**  
Dann bleiben Sie trocken!  
*Then stay dry!*

## Grammatik / Grammar

### Weather Expressions with "es" (it)

**English:** In German, weather expressions use the impersonal pronoun "es" (it). Common patterns include "es ist" + adjective or "es" + verb.

**Examples from conversations:**

- **Es ist sonnig** - It's sunny
- **Es regnet** - It's raining
- **Es schneit** - It's snowing
- **Es ist kalt** - It's cold

### Temperature: "Wie viele Grad?" (How many degrees?)

**English:** To ask about temperature, use "Wie viele Grad sind es?" or "Wie warm ist es?" The answer uses "Es sind" + number + "Grad" or "Es ist" + number + "Grad".

**Examples from conversations:**

- **Wie viele Grad sind es?** - How many degrees is it?
- **Es sind zwanzig Grad** - It's twenty degrees
- **Es sind fünf Grad** - It's five degrees

### Weather Adjectives

**English:** Common weather adjectives include: sonnig (sunny), regnerisch (rainy), windig (windy), kalt (cold), warm (warm), schneereich (snowy).

**Examples from conversations:**

- **Es ist sonnig** - It's sunny
- **Es ist sehr kalt** - It's very cold
- **Es ist warm und sonnig** - It's warm and sunny
- **Der Wind ist stark** - The wind is strong

### Future Weather: "wird" (will be)

**English:** To talk about future weather, use "wird" (will be) + adjective or "wird" + verb infinitive.

**Examples from conversations:**

- **Morgen wird es sonnig** - Tomorrow it will be sunny
- **Übermorgen wird es regnen** - The day after tomorrow it will rain
- **Dann wird es wieder sonnig** - Then it will be sunny again
