# Vegetarische Optionen / Vegetarian Options

---

## 001: Vegetarisches Gericht bestellen / Ordering Vegetarian Dish

**Situation:**  
Anna bestellt ein vegetarisches Gericht im Restaurant.  
*Anna orders a vegetarian dish at the restaurant.*

### Dialog / Dialogue

**Kellner:**  
Was möchten Sie essen?  
*What would you like to eat?*

**Anna:**  
Haben Sie vegetarische Gerichte?  
*Do you have vegetarian dishes?*

**Kellner:**  
Ja, wir haben viele vegetarische Gerichte.  
*Yes, we have many vegetarian dishes.*

**Anna:**  
Was können Sie empfehlen?  
*What can you recommend?*

**Kellner:**  
Das Gemüsecurry ist sehr gut.  
*The vegetable curry is very good.*

**Anna:**  
Das klingt gut. Ich nehme das Gemüsecurry.  
*That sounds good. I'll take the vegetable curry.*

**Kellner:**  
Sehr gut!  
*Very good!*

## Grammatik / Grammar

### Asking "do you have" with "haben Sie" (do you have)

**English:** To ask if a restaurant has vegetarian options, use "haben Sie" (do you have) followed by the item. This is a common way to inquire about menu items.

**Examples from conversations:**

- **Haben Sie vegetarische Gerichte?** - Do you have vegetarian dishes?
- **Haben Sie vegetarische Pizza?** - Do you have vegetarian pizza?
- **Haben Sie einen großen Gemüsesalat?** - Do you have a large vegetable salad?
- **Haben Sie vegetarische Suppe?** - Do you have vegetarian soup?

### Asking "what is" with "was ist" (what is)

**English:** To ask what ingredients are in a dish, use "was ist" (what is) followed by "in" (in) and the dish name. This helps understand what you're ordering.

**Examples from conversations:**

- **Was ist auf der Pizza?** - What is on the pizza?
- **Was ist im Salat?** - What is in the salad?
- **Was ist in der Suppe?** - What is in the soup?
- **Was ist im Menü?** - What is in the menu?

### Expressing "I'll take" with "ich nehme" (I'll take)

**English:** To order a vegetarian dish, use "ich nehme" (I'll take) followed by the item. This is a common way to place an order in German.

**Examples from conversations:**

- **Ich nehme das Gemüsecurry** - I'll take the vegetable curry
- **Ich nehme die Gemüsepfanne** - I'll take the vegetable pan
- **Ich nehme eine Pizza Margherita** - I'll take a Margherita pizza
- **Ich nehme den Salat** - I'll take the salad

### Asking "which" with "welche" (which)

**English:** To ask about different vegetarian options, use "welche" (which) followed by the item type. This helps you choose from available options.

**Examples from conversations:**

- **Welche vegetarischen Gerichte haben Sie?** - Which vegetarian dishes do you have?
- **Welchen Salat möchten Sie?** - Which salad would you like?
- **Welche Beilagen haben Sie?** - Which side dishes do you have?
- **Welche Snacks sind vegetarisch?** - Which snacks are vegetarian?
