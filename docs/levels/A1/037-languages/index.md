# Sprachen / Languages

---

## 001: Welche Sprache sprichst du? / Which Language Do You Speak?

**Situation:**  
Maria fragt Tom, welche Sprache er spricht.  
*Maria asks Tom which language he speaks.*

### Dialog / Dialogue

**Maria:**  
Welche Sprache sprichst du?  
*Which language do you speak?*

**Tom:**  
Ich spreche Deutsch.  
*I speak German.*

**Maria:**  
Nur Deutsch?  
*Only German?*

**Tom:**  
Nein, ich spreche auch Englisch.  
*No, I also speak English.*

**Maria:**  
Das ist gut.  
*That's good.*

**Tom:**  
Und du? Welche Sprache sprichst du?  
*And you? Which language do you speak?*

**Maria:**  
Ich spreche Deutsch und Spanisch.  
*I speak German and Spanish.*

**Tom:**  
Das sind zwei Sprachen.  
*Those are two languages.*

**Maria:**  
Ja, ich lerne noch mehr.  
*Yes, I'm learning more.*

## 002: Deutsch lernen / Learning German

Anna spricht mit Max über das Deutschlernen.  
*Anna talks with Max about learning German.*

**Max:**

*For one year.*

**Anna:**

*That's good. How do you find it?*

**Max:**  
Es ist schwer, aber interessant.  
*It's difficult, but interesting.*

**Anna:**  
Das verstehe ich.  
*I understand that.*

**Max:**  
Ich übe jeden Tag.  
*I practice every day.*

**Anna:**  
Das ist wichtig.  
*That's important.*

## Grammatik / Grammar

### Asking About Languages with "welche Sprache"

**English:** To ask which language someone speaks, use "welche Sprache sprichst du?" (which language do you speak?) or "wie viele Sprachen sprichst du?" (how many languages do you speak?). These questions help understand language abilities.

**Examples from conversations:**

- **Welche Sprache sprichst du?** - Which language do you speak?
- **Wie viele Sprachen sprichst du?** - How many languages do you speak?
- **Was ist deine Muttersprache?** - What is your native language?
- **Welche Fremdsprache lernst du?** - Which foreign language are you learning?

### Expressing Language Ability with "sprechen" and "können"

**English:** To say what languages you speak, use "ich spreche" (I speak) followed by the language. To express ability, use "ich kann" (I can) with the language or activity.

**Examples from conversations:**

- **Ich spreche Deutsch** - I speak German
- **Ich spreche auch Englisch** - I also speak English
- **Ich kann übersetzen** - I can translate
- **Ich spreche drei Sprachen** - I speak three languages

### Learning Languages with "lernen" and "üben"

**English:** To talk about learning languages, use "lernen" (to learn) for the learning process and "üben" (to practice) for practicing. Use "seit" (since/for) to express how long you've been learning.

**Examples from conversations:**

- **Ich lerne Deutsch** - I'm learning German
- **Ich übe jeden Tag** - I practice every day
- **Seit einem Jahr** - For one year
- **Wie übst du Deutsch?** - How do you practice German?

### Describing Difficulty with "schwer" and "schwierig"

**English:** To describe how difficult a language is, use "schwer" (difficult) or "schwierig" (difficult). Use "weil" (because) to explain why something is difficult.

**Examples from conversations:**

- **Es ist schwer, aber interessant** - It's difficult, but interesting
- **Ich finde Deutsch schwierig** - I find German difficult
- **Weil die Grammatik kompliziert ist** - Because the grammar is complicated
- **Weil die Schriftzeichen schwer sind** - Because the characters are difficult
