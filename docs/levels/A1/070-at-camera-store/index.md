# Im Fotogeschäft / At Camera Store

---

## 001: Kamera kaufen / Buying Camera

**Situation:**  
Tom kauft eine Kamera.  
*Tom buys a camera.*

### Dialog / Dialogue

**Tom:**  
Guten Tag! Ich suche eine Kamera.  
*Good day! I'm looking for a camera.*

**Verkäufer:**  
Guten Tag! Welche Art Kamera möchten Sie?  
*Good day! What kind of camera would you like?*

**Tom:**  
Eine digitale Kamera.  
*A digital camera.*

**Verkäufer:**  
Hier sind verschiedene Kameras.  
*Here are different cameras.*

**Tom:**  
Wie viel kostet diese hier?  
*How much does this one cost?*

**Verkäufer:**  
Sie kostet dreihundert Euro.  
*It costs three hundred euros.*

**Tom:**  
Das ist teuer. Haben Sie auch günstigere?  
*That's expensive. Do you also have cheaper ones?*

**Verkäufer:**  
Ja, hier ist eine für zweihundert Euro.  
*Yes, here is one for two hundred euros.*

**Tom:**  
Das ist besser. Kann ich sie ausprobieren?  
*That's better. Can I try it out?*

**Verkäufer:**  
Natürlich!  
*Of course!*

**Tom:**  
Sie funktioniert gut. Ich nehme sie.  
*It works well. I'll take it.*

## Grammatik / Grammar

### Asking "For Which Camera": "Für welche Kamera?" (For which camera?)

**English:** To ask what camera something is for, use "Für welche Kamera" followed by the item.

**Examples from conversations:**

- **Für welche Kamera?** - For which camera?
- **Für eine Canon-Kamera** - For a Canon camera
- **Für eine Nikon-Kamera** - For a Nikon camera

### Expressing Photo Size: "Zehn mal fünfzehn Zentimeter" (Ten by fifteen centimeters)

**English:** To express dimensions or size, use numbers followed by "mal" (by) and the unit of measurement.

**Examples from conversations:**

- **Zehn mal fünfzehn Zentimeter** - Ten by fifteen centimeters
- **Welche Größe?** - What size?
- **Ein mittelgroßes Album** - A medium-sized album

### Asking About Duration for Services: "Wie lange dauert es?" (How long does it take?)

**English:** To ask how long a service takes, use "Wie lange dauert es" or "Wie lange dauert" followed by the service.

**Examples from conversations:**

- **Wie lange dauert es?** - How long does it take?
- **Ungefähr eine Stunde** - Approximately one hour
- **Ungefähr zwei Wochen** - Approximately two weeks

### Expressing Capacity: "Gigabyte" (Gigabyte)

**English:** "Gigabyte" is used to express storage capacity for digital devices like memory cards.

**Examples from conversations:**

- **Zweiunddreißig Gigabyte** - Thirty-two gigabytes
- **Welche Größe?** - What size?
- **Eine Speicherkarte kostet fünfundzwanzig Euro** - A memory card costs twenty-five euros
