# In der Bank / At the Bank

---

## 001: Konto eröffnen / Opening an Account

**Situation:**  
Julia möchte ein Bankkonto eröffnen.  
*Julia wants to open a bank account.*

### Dialog / Dialogue

**Julia:**  
Guten Tag. Ich möchte ein Konto eröffnen.  
*Good day. I'd like to open an account.*

**Bankangestellte:**  
Gut. Haben Sie Ihren Ausweis dabei?  
*Good. Do you have your ID with you?*

**Julia:**  
Ja, hier ist mein Ausweis.  
*Yes, here is my ID.*

**Bankangestellte:**  
Sehr gut. Füllen Sie bitte dieses Formular aus.  
*Very good. Please fill out this form.*

**Julia:**  
Gut.  
*Good.*

**Bankangestellte:**  
Welche Art von Konto möchten Sie?  
*What type of account would you like?*

**Julia:**  
Ein Girokonto, bitte.  
*A checking account, please.*

**Bankangestellte:**  
Sehr gut. Das Konto ist kostenlos.  
*Very good. The account is free.*

**Julia:**  
Das ist gut.  
*That's good.*

## Grammatik / Grammar

### Banking Actions with "möchte" (would like)

**English:** Common banking actions use "Ich möchte..." (I would like...) followed by the action. This is polite and appropriate for A1 level.

**Examples from conversations:**

- **Ich möchte ein Konto eröffnen** - I'd like to open an account
- **Ich möchte Geld abheben** - I'd like to withdraw money
- **Ich möchte Geld einzahlen** - I'd like to deposit money
- **Ich möchte Geld überweisen** - I'd like to transfer money

### Asking "How much?" with "Wie viel"

**English:** Use "Wie viel möchten Sie...?" (How much would you like...?) to ask about amounts in banking contexts.

**Examples from conversations:**

- **Wie viel möchten Sie abheben?** - How much would you like to withdraw?
- **Wie viel möchten Sie einzahlen?** - How much would you like to deposit?
- **Wie viel möchten Sie überweisen?** - How much would you like to transfer?
- **Wie viel möchten Sie wechseln?** - How much would you like to exchange?

### Banking Vocabulary

**English:** Common banking terms include "Konto" (account), "Girokonto" (checking account), "Kreditkarte" (credit card), "PIN" (PIN), "Kontoauszug" (account statement), and "Überweisung" (transfer).

**Examples from conversations:**

- **Ich möchte ein Konto eröffnen** - I'd like to open an account
- **Ein Girokonto, bitte** - A checking account, please
- **Ich möchte eine Kreditkarte beantragen** - I'd like to apply for a credit card
- **Ich brauche einen Kontoauszug** - I need an account statement

### Filling Out Forms

**English:** At the bank, you often need to fill out forms. Use "Füllen Sie bitte dieses Formular aus" (Please fill out this form).

**Examples from conversations:**

- **Füllen Sie bitte dieses Formular aus** - Please fill out this form
- **Ich fülle das Formular aus** - I'm filling out the form
- **Füllen Sie bitte dieses Formular aus** - Please fill out this form
