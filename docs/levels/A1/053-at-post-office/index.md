# Auf der Post / At the Post Office

---

## 001: Brief abschicken / Sending a Letter

**Situation:**  
Maria möchte einen Brief abschicken.  
*Maria wants to send a letter.*

### Dialog / Dialogue

**Maria:**  
Guten Tag. Ich möchte einen Brief abschicken.  
*Good day. I'd like to send a letter.*

**Postangestellte:**  
Gut. Wohin möchten Sie den Brief schicken?  
*Good. Where would you like to send the letter?*

**Maria:**  
Nach Deutschland.  
*To Germany.*

**Postangestellte:**  
Gut. Das kostet einen Euro.  
*Good. That costs one euro.*

**Maria:**  
Gut. Hier ist das Geld.  
*Good. Here's the money.*

**Postangestellte:**  
Danke. Hier ist die Briefmarke.  
*Thanks. Here's the stamp.*

**Maria:**  
Danke schön.  
*Thank you very much.*

**Postangestellte:**  
Gern geschehen.  
*You're welcome.*

## Grammatik / Grammar

### Asking "Where to?" with "Wohin möchten Sie... schicken?"

**English:** When sending mail or packages, use "Wohin möchten Sie... schicken?" (Where would you like to send...?) to ask about the destination.

**Examples from conversations:**

- **Wohin möchten Sie den Brief schicken?** - Where would you like to send the letter?
- **Nach Deutschland** - To Germany
- **Wohin möchten Sie das Paket schicken?** - Where would you like to send the package?
- **Nach Österreich** - To Austria

### Expressing Weight with "wie schwer"

**English:** To ask about weight, use "Wie schwer ist...?" (How heavy is...?) or "wie schwer" (how heavy). Answer with "Kilo" (kilo) or "Gramm" (gram).

**Examples from conversations:**

- **Wie schwer ist das Paket?** - How heavy is the package?
- **Zwei Kilo** - Two kilos
- **Das Paket wiegt zwei Kilo** - The package weighs two kilos

### Postal Services Vocabulary

**English:** Common postal service terms include "Brief" (letter), "Paket" (package), "Briefmarke" (stamp), "Einschreiben" (registered mail), "Postkarte" (postcard), and "Postfach" (PO box).

**Examples from conversations:**

- **Ich möchte einen Brief abschicken** - I'd like to send a letter
- **Ich brauche Briefmarken** - I need stamps
- **Ich möchte ein Paket abschicken** - I'd like to send a package
- **Ich möchte einen Brief als Einschreiben abschicken** - I'd like to send a letter as registered mail

### Filling Out Forms with "Formular ausfüllen"

**English:** When you need to fill out forms at the post office, use "Füllen Sie bitte dieses Formular aus" (Please fill out this form) or "Formular ausfüllen" (to fill out a form).

**Examples from conversations:**

- **Füllen Sie bitte dieses Formular aus** - Please fill out this form
- **Ich fülle das Formular aus** - I'm filling out the form
- **Füllen Sie bitte dieses Formular aus** - Please fill out this form
