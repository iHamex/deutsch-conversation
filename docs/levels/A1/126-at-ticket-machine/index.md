# Am Fahrkartenautomaten / At Ticket Machine

---

## 001: Erste Nutzung / First Use

**Situation:**  
Anna steht zum ersten Mal vor einem Fahrkartenautomaten und braucht Hilfe.  
*Anna is standing in front of a ticket machine for the first time and needs help.*

### Dialog / Dialogue

**Anna:**  
Entschuldigung, wie funktioniert dieser Automat?  
*Excuse me, how does this machine work?*

**Andere Person:**  
Sie wählen zuerst Ihr Ziel aus.  
*You first select your destination.*

**Anna:**  
Wo wähle ich das Ziel?  
*Where do I select the destination?*

**Andere Person:**  
Sie drücken auf den Bildschirm und wählen Ihr Ziel.  
*You press on the screen and select your destination.*

**Anna:**  
Und dann?  
*And then?*

**Andere Person:**  
Dann wählen Sie die Art der Fahrkarte.  
*Then you select the type of ticket.*

**Anna:**  
Ich möchte eine Einzelfahrkarte.  
*I would like a single ticket.*

**Andere Person:**  
Dann drücken Sie auf "Einzelfahrkarte".  
*Then you press on "single ticket".*

**Anna:**  
Verstanden. Vielen Dank für Ihre Hilfe!  
*Understood. Thank you very much for your help!*

## 002: Mit Münzen bezahlen / Paying with Coins

**Situation:**  
Tom möchte am Automaten eine Fahrkarte mit Münzen kaufen.  
*Tom wants to buy a ticket at the machine with coins.*

### Dialog / Dialogue

**Tom:**  
Entschuldigung, nimmt dieser Automat Münzen?  
*Excuse me, does this machine accept coins?*

**Angestellter:**  
Ja, der Automat nimmt Münzen und Scheine.  
*Yes, the machine accepts coins and bills.*

**Tom:**  
Gut. Ich habe nur Münzen.  
*Good. I only have coins.*

**Angestellter:**  
Das ist kein Problem. Sie werfen die Münzen hier ein.  
*That's no problem. You insert the coins here.*

**Tom:**  
Welche Münzen nimmt der Automat?  
*Which coins does the machine accept?*

**Angestellter:**  
Der Automat nimmt alle Euro-Münzen.  
*The machine accepts all euro coins.*

**Tom:**  
Perfekt. Dann kaufe ich jetzt meine Fahrkarte.  
*Perfect. Then I'll buy my ticket now.*

**Angestellter:**  
Viel Erfolg!  
*Good luck!*

**Tom:**  
Danke!  
*Thanks!*

*What is the problem?*

*The screen doesn't show anything.*

*Is there another machine?*

**Angestellter:**  
Ja, dort drüben ist ein weiterer Automat.  
*Yes, over there is another machine.*

**Angestellter:**

*Yes, of course. The counter is there.*

**Sarah:**  
Danke für die Information!  
*Thanks for the information!*

## Grammatik / Grammar

### Asking "wie" (how)

**English:** "Wie" asks about manner or method. It's used when asking how something works or how to do something.

**Examples from conversations:**

- **Wie funktioniert dieser Automat?** - How does this machine work?
- **Wie funktioniert das?** - How does that work?
- **Wie kann ich die Sprache ändern?** - How can I change the language?

### Using "können" (can/to be able)

**English:** "Können" expresses ability or possibility. It's very common in questions asking for permission or help.

**Examples from conversations:**

- **Kann ich mit Karte bezahlen?** - Can I pay with card?
- **Können Sie mir helfen?** - Can you help me?
- **Kann ich die Karte umtauschen?** - Can I exchange the ticket?

### Expressing problems with "funktioniert nicht" (doesn't work)

**English:** "Funktioniert nicht" means "doesn't work" and is used to describe problems with machines or devices.

**Examples from conversations:**

- **Dieser Automat funktioniert nicht** - This machine doesn't work
- **Der Bildschirm zeigt nichts an** - The screen doesn't show anything
- **Die Fahrkarte kommt nicht heraus** - The ticket doesn't come out

### Giving instructions with "Sie" (you - formal)

**English:** When giving instructions, use the formal "Sie" form. The verb comes after "Sie" in commands.

**Examples from conversations:**

- **Sie wählen zuerst Ihr Ziel aus** - You first select your destination
- **Sie drücken auf den Bildschirm** - You press on the screen
- **Sie stecken die Karte hier ein** - You insert the card here
