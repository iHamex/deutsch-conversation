# Telefonieren / Making Phone Calls

---

## 001: Telefonnummer wählen / Dialing Phone Number

**Situation:**  
Anna wählt eine Telefonnummer.  
*Anna dials a phone number.*

### Dialog / Dialogue

**Anna:**  
Ich muss telefonieren.  
*I need to make a phone call.*

**Freundin:**  
Welche Nummer rufst du an?  
*Which number are you calling?*

**Anna:**  
Die Nummer meiner Mutter.  
*My mother's number.*

**Freundin:**  
Kennst du die Nummer?  
*Do you know the number?*

**Anna:**  
Ja, ich kenne sie.  
*Yes, I know it.*

**Freundin:**  
Dann ruf einfach an.  
*Then just call.*

**Anna:**  
Okay, ich rufe jetzt an.  
*Okay, I'm calling now.*

## 002: Telefonat beginnen / Starting Phone Call

**Situation:**  
Tom beginnt ein Telefonat.  
*Tom starts a phone call.*

### Dialog / Dialogue

**Tom:**  
Hallo?  
*Hello?*

**Person:**  
Hallo, wer ist da?  
*Hello, who is there?*

**Tom:**  
Hier ist Tom.  
*This is Tom.*

**Person:**  
Ah, Tom! Wie geht es dir?  
*Ah, Tom! How are you?*

**Tom:**  
Gut, danke. Und dir?  
*Good, thanks. And you?*

**Person:**  
Auch gut.  
*Also good.*

**Tom:**  
Ich rufe wegen morgen an.  
*I'm calling about tomorrow.*

## Grammatik / Grammar

### Expressing "I need to call" with "ich muss telefonieren" (I need to make a phone call)

**English:** To express that you need to make a phone call, use "ich muss telefonieren" (I need to make a phone call) or "ich rufe an" (I'm calling). This initiates phone communication.

**Examples from conversations:**

- **Ich muss telefonieren** - I need to make a phone call
- **Ich rufe jetzt an** - I'm calling now
- **Welche Nummer rufst du an?** - Which number are you calling?
- **Ich rufe wegen morgen an** - I'm calling about tomorrow

### Asking "is someone there" with "ist ... da" (is ... there)

**English:** To ask if someone is available on the phone, use "ist [name] da" (is [name] there). This is the standard way to ask for someone.

**Examples from conversations:**

- **Ist Max da?** - Is Max there?
- **Ist Anna da?** - Is Anna there?
- **Nein, sie ist nicht da** - No, she's not here
- **Einen Moment, ich hole ihn** - One moment, I'll get him

### Expressing "this is" with "hier ist" (this is)

**English:** To identify yourself on the phone, use "hier ist" (this is) followed by your name. This is how Germans introduce themselves on calls.

**Examples from conversations:**

- **Hier ist Tom** - This is Tom
- **Hier ist Max** - This is Max
- **Hier ist Maria** - This is Maria
- **Hallo, hier ist Anna** - Hello, this is Anna

### Expressing "I need to hang up" with "ich muss auflegen" (I need to hang up)

**English:** To end a phone call, use "ich muss auflegen" (I need to hang up) or say goodbye with "bis bald" (see you soon) or "tschüss" (bye).

**Examples from conversations:**

- **Ich muss jetzt auflegen** - I need to hang up now
- **Bis bald!** - See you soon!
- **Tschüss!** - Bye!
- **Auf Wiedersehen!** - Goodbye!
