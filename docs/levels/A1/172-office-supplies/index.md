# Büromaterial / Office Supplies

## 001: Stifte kaufen / Buying Pens

**Situation:**  
Anna kauft Stifte.  
*Anna buys pens.*

### Dialog / Dialogue

**Anna:**  
Ich brauche Stifte.  
*I need pens.*

**Verkäufer:**  
Welche Stifte möchten Sie?  
*Which pens would you like?*

**Anna:**  
Kugelschreiber, bitte.  
*Ballpoint pens, please.*

**Verkäufer:**  
Wie viele möchten Sie?  
*How many would you like?*

**Anna:**  
Zehn Stifte.  
*Ten pens.*

**Verkäufer:**  
Das macht fünf Euro.  
*That's five euros.*

**Anna:**  
Hier ist das Geld.  
*Here is the money.*

## Grammatik / Grammar

### Expressing "I need" with "ich brauche" (I need)

**English:** To express that you need office supplies, use "ich brauche" (I need) followed by the item. This is the standard way to request supplies.

**Examples from conversations:**

- **Ich brauche Stifte** - I need pens
- **Ich brauche Papier** - I need paper
- **Ich brauche Ordner** - I need folders
- **Ich brauche einen Hefter** - I need a stapler

### Asking "which" with "welche" (which)

**English:** To ask which type or color of office supply someone wants, use "welche" (which) followed by the item. This helps identify preferences.

**Examples from conversations:**

- **Welche Stifte möchten Sie?** - Which pens would you like?
- **Welche Farbe möchten Sie?** - Which color would you like?
- **Welche Größe möchten Sie?** - Which size would you like?
- **Welche Art von Klebestreifen?** - What kind of tape?

### Asking "how many" with "wie viele" (how many)

**English:** To ask how many items someone wants, use "wie viele" (how many) followed by the item. This helps determine quantity.

**Examples from conversations:**

- **Wie viele möchten Sie?** - How many would you like?
- **Wie viele Blätter?** - How many sheets?
- **Wie viele Ordner?** - How many folders?
- **Wie viele Rollen?** - How many rolls?

### Expressing "I'll take" with "ich nehme" (I'll take)

**English:** To confirm a purchase, use "ich nehme" (I'll take) followed by the item or "ich nehme es/sie" (I'll take it/them). This completes the purchase.

**Examples from conversations:**

- **Ich nehme es** - I'll take it
- **Ich nehme diesen** - I'll take this one
- **Ich nehme ihn** - I'll take it
- **Ich nehme sie** - I'll take them
