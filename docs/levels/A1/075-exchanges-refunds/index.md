# Umtausch und Rückerstattung / Exchanges and Refunds

---

## 001: Größe umtauschen / Exchanging Size

**Situation:**  
Anna tauscht ein Hemd gegen eine andere Größe um.  
*Anna exchanges a shirt for a different size.*

### Dialog / Dialogue

**Anna:**  
Guten Tag! Ich möchte dieses Hemd umtauschen.  
*Good day! I would like to exchange this shirt.*

**Verkäuferin:**  
Guten Tag! Was ist das Problem?  
*Good day! What is the problem?*

**Anna:**  
Es ist zu klein. Ich brauche eine größere Größe.  
*It's too small. I need a larger size.*

**Verkäuferin:**  
Welche Größe brauchen Sie?  
*What size do you need?*

**Anna:**  
Größe 42, bitte.  
*Size 42, please.*

**Verkäuferin:**  
Haben Sie den Kassenzettel?  
*Do you have the receipt?*

**Anna:**  
Ja, hier ist er.  
*Yes, here it is.*

**Verkäuferin:**  
Gut. Hier ist Größe 42.  
*Good. Here is size 42.*

**Anna:**  
Danke schön!  
*Thank you very much!*

## Grammatik / Grammar

### Expressing Exchange: "umtauschen" (to exchange)

**English:** "Umtauschen" means "to exchange" something for another item. It's a separable verb, so "um" goes to the end in main clauses.

**Examples from conversations:**

- **Ich möchte dieses Hemd umtauschen** - I would like to exchange this shirt
- **Ich kann es umtauschen** - I can exchange it
- **Wie lange kann ich etwas umtauschen?** - How long can I exchange something?

### Expressing Preference: "gefallen" (to like/to please)

**English:** "Gefallen" means "to like" or "to please" and is used with the dative case. The thing that is liked is the subject.

**Examples from conversations:**

- **Die Farbe gefällt mir nicht** - I don't like the color
- **Es gefällt mir nicht** - I don't like it
- **Welche Farbe möchten Sie?** - What color would you like?

### Expressing Payment Method: "Wie haben Sie bezahlt?" (How did you pay?)

**English:** To ask how someone paid, use "Wie haben Sie bezahlt" followed by a question mark.

**Examples from conversations:**

- **Wie haben Sie bezahlt?** - How did you pay?
- **Mit Karte** - With card
- **Ich zahle die Differenz** - I'll pay the difference

### Expressing Time Period: "Sie haben... Zeit" (You have... time)

**English:** To express how much time someone has for something, use "Sie haben" followed by the time period and "Zeit" (time).

**Examples from conversations:**

- **Sie haben vierzehn Tage Zeit** - You have fourteen days
- **Wie lange dauert das?** - How long does that take?
- **Ungefähr drei bis fünf Tage** - Approximately three to five days
