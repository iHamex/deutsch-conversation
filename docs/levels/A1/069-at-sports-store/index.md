# Im Sportgeschäft / At Sports Store

---

## 001: Sportschuhe kaufen / Buying Sports Shoes

**Situation:**  
Tom kauft Sportschuhe für das Laufen.  
*Tom buys sports shoes for running.*

### Dialog / Dialogue

**Tom:**  
Guten Tag! Ich suche Sportschuhe.  
*Good day! I'm looking for sports shoes.*

**Verkäufer:**  
Guten Tag! Für welchen Sport?  
*Good day! For which sport?*

**Tom:**  
Für das Laufen.  
*For running.*

**Verkäufer:**  
Welche Größe haben Sie?  
*What size do you have?*

**Tom:**  
Größe 43.  
*Size 43.*

**Verkäufer:**  
Hier sind Laufschuhe in Ihrer Größe.  
*Here are running shoes in your size.*

**Tom:**  
Kann ich diese hier anprobieren?  
*Can I try on this one?*

**Verkäufer:**  
Natürlich!  
*Of course!*

**Tom:**  
Die Schuhe passen gut. Wie viel kosten sie?  
*The shoes fit well. How much do they cost?*

**Verkäufer:**  
Sie kosten neunundachtzig Euro.  
*They cost eighty-nine euros.*

**Tom:**  
Okay, ich nehme sie.  
*Okay, I'll take them.*

## Grammatik / Grammar

### Asking "For Which Sport": "Für welchen Sport?" (For which sport?)

**English:** To ask what sport something is for, use "Für welchen Sport" followed by the item or action.

**Examples from conversations:**

- **Für welchen Sport?** - For which sport?
- **Für das Laufen** - For running
- **Für Anfänger oder Fortgeschrittene?** - For beginners or advanced?

### Expressing Both Items: "beides" (both)

**English:** "Beides" means "both" and is used when referring to two items or options together.

**Examples from conversations:**

- **Ich nehme beides** - I'll take both
- **Eine Sporthose und ein T-Shirt** - Sports pants and a T-shirt
- **Wie viel kostet beides?** - How much do both cost?

### Asking About Weight: "Welches Gewicht?" (What weight?)

**English:** To ask about the weight of something, use "Welches Gewicht" followed by "möchten Sie" (would you like).

**Examples from conversations:**

- **Welches Gewicht?** - What weight?
- **Zwei Kilogramm, bitte** - Two kilograms, please
- **Zwei Hanteln kosten dreißig Euro** - Two dumbbells cost thirty euros

### Expressing Size: "Größe..." (Size...)

**English:** To express size, use "Größe" followed by the size number or letter.

**Examples from conversations:**

- **Welche Größe haben Sie?** - What size do you have?
- **Größe 43** - Size 43
- **Größe L, bitte** - Size L, please
- **Größe 5, bitte** - Size 5, please
- **Sie brauchen Größe M** - You need size M
