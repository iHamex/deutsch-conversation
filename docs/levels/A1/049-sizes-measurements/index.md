# Größen und Maße / Sizes and Measurements

---

## 001: Kleidergröße fragen / Asking About Clothing Size

**Situation:**  
Maria fragt nach ihrer Kleidergröße in einem Geschäft.  
*Maria asks about her clothing size in a store.*

### Dialog / Dialogue

**Maria:**  
Guten Tag. Welche Größe habe ich?  
*Good day. What size do I have?*

**Verkäuferin:**  
Gut. Lassen Sie mich messen.  
*Good. Let me measure.*

**Maria:**  
Danke.  
*Thanks.*

**Verkäuferin:**  
Sie haben Größe 38.  
*You have size 38.*

**Maria:**  
Gut. Und welche Größe ist das in anderen Ländern?  
*Good. And what size is that in other countries?*

**Verkäuferin:**  
Größe 38 ist Größe M in den USA.  
*Size 38 is size M in the USA.*

**Maria:**  
Verstanden. Danke schön.  
*Understood. Thank you very much.*

**Verkäuferin:**  
Gern geschehen.  
*You're welcome.*

*"One size" means there's only one size for everyone.*

**Markus:**

*Good to know. Thanks.*

**Verkäufer:**  
Gern geschehen.  
*You're welcome.*

## Grammatik / Grammar

### Asking About Size with "Welche Größe"

**English:** To ask about size, use "Welche Größe..." (What size...) followed by "haben Sie" (do you have) or "ist..." (is...).

**Examples from conversations:**

- **Welche Größe habe ich?** - What size do I have?
- **Welche Größen haben Sie?** - What sizes do you have?
- **Welche Größe hat Ihre Freundin?** - What size does your girlfriend have?
- **Welche Größe ist Größe 40 in den USA?** - What size is size 40 in the USA?

### Comparing Sizes with "größer als" and "kleiner als"

**English:** To compare sizes, use "größer als" (larger than) or "kleiner als" (smaller than). "Zu klein" means "too small" and "zu groß" means "too big."

**Examples from conversations:**

- **Größe L ist größer als Größe M** - Size L is larger than size M
- **Größe L ist zu klein** - Size L is too small
- **Größe XL passt besser** - Size XL fits better

### Measurements with "wiegen" and "lang/breit"

**English:** Use "wiegen" (to weigh) for weight and "lang" (long) and "breit" (wide) for dimensions. Numbers are followed by units like "Kilo" (kilo) or "Zentimeter" (centimeter).

**Examples from conversations:**

- **Wie schwer ist dieses Produkt?** - How heavy is this product?
- **Dieses Produkt wiegt zwei Kilo** - This product weighs two kilos
- **Es ist dreißig Zentimeter lang** - It's thirty centimeters long
- **Zwanzig Zentimeter breit** - Twenty centimeters wide

### Expressing Fit with "passen"

**English:** The verb "passen" (to fit) is used to describe how clothing or items fit. Use "passt besser" (fits better) or "passt gut" (fits well).

**Examples from conversations:**

- **Wie passt es?** - How does it fit?
- **Größe M passt besser** - Size M fits better
- **Größe XL passt besser** - Size XL fits better
- **Wie passen die Größen?** - How do the sizes fit?
