# Artikel zurückgeben / Returning Items

---

## 001: Falsche Größe zurückgeben / Returning Wrong Size

**Situation:**  
Anna gibt ein Hemd zurück, weil es nicht passt.  
*Anna returns a shirt because it doesn't fit.*

### Dialog / Dialogue

**Anna:**  
Guten Tag! Ich möchte dieses Hemd zurückgeben.  
*Good day! I would like to return this shirt.*

**Verkäuferin:**  
Guten Tag! Was ist das Problem?  
*Good day! What is the problem?*

**Anna:**  
Es passt nicht. Es ist zu klein.  
*It doesn't fit. It's too small.*

**Verkäuferin:**  
Haben Sie den Kassenzettel?  
*Do you have the receipt?*

**Anna:**  
Ja, hier ist der Kassenzettel.  
*Yes, here is the receipt.*

**Verkäuferin:**  
Gut. Wann haben Sie es gekauft?  
*Good. When did you buy it?*

**Anna:**  
Vor drei Tagen.  
*Three days ago.*

**Verkäuferin:**  
Okay, ich gebe Ihnen das Geld zurück.  
*Okay, I'll give you the money back.*

**Anna:**  
Danke schön!  
*Thank you very much!*

## Grammatik / Grammar

### Expressing Return: "zurückgeben" (to return)

**English:** "Zurückgeben" means "to return" something to a store. It's a separable verb, so "zurück" goes to the end in main clauses.

**Examples from conversations:**

- **Ich möchte dieses Hemd zurückgeben** - I would like to return this shirt
- **Ich gebe Ihnen das Geld zurück** - I'll give you the money back
- **Ich kann es zurückgeben** - I can return it

### Asking About Receipt: "Haben Sie den Kassenzettel?" (Do you have the receipt?)

**English:** To ask if someone has a receipt, use "Haben Sie den Kassenzettel" followed by a question mark.

**Examples from conversations:**

- **Haben Sie den Kassenzettel?** - Do you have the receipt?
- **Ja, hier ist der Kassenzettel** - Yes, here is the receipt
- **Nein, ich habe keinen Kassenzettel** - No, I don't have a receipt

### Expressing Problems: "Was ist das Problem?" (What is the problem?)

**English:** To ask what the problem is, use "Was ist das Problem" followed by a question mark.

**Examples from conversations:**

- **Was ist das Problem?** - What is the problem?
- **Es passt nicht** - It doesn't fit
- **Es funktioniert nicht** - It doesn't work
- **Die Farbe ist falsch** - The color is wrong

### Expressing Time: "vor" (ago)

**English:** "Vor" means "ago" and is used with time expressions to indicate when something happened in the past.

**Examples from conversations:**

- **Vor drei Tagen** - Three days ago
- **Vor einem Monat** - One month ago
- **Wann haben Sie es gekauft?** - When did you buy it?
