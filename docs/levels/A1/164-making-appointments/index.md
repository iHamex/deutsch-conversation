# Termine vereinbaren / Making Appointments

---

## 001: Arzttermin vereinbaren / Making Doctor Appointment

**Situation:**  
Anna vereinbart einen Arzttermin.  
*Anna makes a doctor appointment.*

### Dialog / Dialogue

**Anna:**  
Guten Tag, ich möchte einen Termin vereinbaren.  
*Good day, I'd like to make an appointment.*

**Rezeptionistin:**  
Gerne. Wann passt es Ihnen?  
*Gladly. When is convenient for you?*

**Anna:**  
Am besten heute Nachmittag.  
*Best this afternoon.*

**Rezeptionistin:**  
Heute um drei Uhr ist noch frei.  
*Today at three o'clock is still available.*

**Anna:**  
Das passt gut.  
*That works well.*

**Rezeptionistin:**  
Wie ist Ihr Name?  
*What is your name?*

**Anna:**  
Anna Müller.  
*Anna Müller.*

**Rezeptionistin:**  
Gut, Sie sind angemeldet.  
*Good, you are registered.*

*Maria reserves a table at the restaurant.*

**Maria:**

**Kellner:**

*Klaus reschedules an appointment.*

*Tomorrow at two o'clock.*

**Rezeptionistin:**  
Möchten Sie den Termin verschieben?  
*Would you like to reschedule the appointment?*

**Klaus:**  
Ja, bitte.  
*Yes, please.*

**Rezeptionistin:**  
Wann passt es Ihnen?  
*When is convenient for you?*

**Klaus:**  
Nächste Woche Dienstag.  
*Next week Tuesday.*

**Rezeptionistin:**  
Gut, ich ändere den Termin.  
*Good, I'll change the appointment.*

## Grammatik / Grammar

### Expressing "I'd like to make an appointment" with "ich möchte einen Termin vereinbaren" (I'd like to make an appointment)

**English:** To request an appointment, use "ich möchte einen Termin vereinbaren" (I'd like to make an appointment). This is the standard way to start booking appointments.

**Examples from conversations:**

- **Ich möchte einen Termin vereinbaren** - I'd like to make an appointment
- **Ich brauche einen Termin** - I need an appointment
- **Ich möchte einen Tisch reservieren** - I'd like to reserve a table
- **Wann passt es Ihnen?** - When is convenient for you?

### Asking "when" with "wann" (when)

**English:** To ask when someone wants an appointment, use "wann" (when) followed by "passt es Ihnen" (is convenient for you) or "möchten Sie" (would you like).

**Examples from conversations:**

- **Wann passt es Ihnen?** - When is convenient for you?
- **Wann möchten Sie kommen?** - When would you like to come?
- **Für wann?** - For when?
- **Wann haben Sie einen Termin?** - When do you have an appointment?

### Expressing "that works" with "das passt" (that works)

**English:** To confirm that a time works for you, use "das passt" (that works) or "das passt gut" (that works well). This is how you accept appointment times.

**Examples from conversations:**

- **Das passt gut** - That works well
- **Das passt** - That works
- **Am besten heute Nachmittag** - Best this afternoon
- **Heute um drei Uhr ist noch frei** - Today at three o'clock is still available

### Asking "for how many" with "für wie viele" (for how many)

**English:** To ask how many people an appointment or reservation is for, use "für wie viele Personen" (for how many people). This is common for restaurant reservations.

**Examples from conversations:**

- **Für wie viele Personen?** - For how many people?
- **Für zwei Personen** - For two people
- **Für heute Abend um sieben Uhr** - For this evening at seven o'clock
- **Der Tisch ist reserviert** - The table is reserved
